Frontend Development Prompt for CDSS Clinical Decision Support System
Project Context
Build a production-grade, interactive web frontend for a Clinical Decision Support System (CDSS) that integrates with an existing Azure-hosted FastAPI backend. The system orchestrates 5 specialized AI agents to provide evidence-based clinical recommendations.

Backend Architecture (Already Deployed)
API Base: FastAPI REST API on Azure App Service

Authentication: Azure Entra ID (OAuth 2.0)

Key Endpoints:

POST /api/v1/query - Submit clinical queries

POST /api/v1/query/stream - Streaming responses (SSE)

GET /api/v1/patients/{id} - Patient profiles

POST /api/v1/drugs/interactions - Drug interaction checks

POST /api/v1/search/literature - PubMed search

POST /api/v1/documents/ingest - Document upload

GET /api/v1/audit - Audit trail (admin only)

Frontend Requirements
1. Technology Stack
Framework: React 18+ with TypeScript

UI Library: Material-UI (MUI) v5 or Ant Design (medical-grade aesthetics)

State Management: React Query (TanStack Query) for API calls + Zustand for global state

Routing: React Router v6

Forms: React Hook Form + Zod validation

Charts: Recharts or Chart.js for lab results visualization

Real-time: EventSource API for SSE streaming responses

Build Tool: Vite

Styling: Tailwind CSS + component library

API Client Setup
// src/lib/api-client.ts
import axios from 'axios';
import { msalInstance } from './auth';

const apiClient = axios.create({
  baseURL: process.env.VITE_API_BASE_URL,
  timeout: 30000,
});

apiClient.interceptors.request.use(async (config) => {
  const account = msalInstance.getAllAccounts()[0];
  const response = await msalInstance.acquireTokenSilent({
    scopes: [`${process.env.VITE_API_SCOPE}`],
    account,
  });
  config.headers.Authorization = `Bearer ${response.accessToken}`;
  return config;
});

Streaming Response Handler
// src/hooks/useStreamingQuery.ts
const useStreamingQuery = (query: string, patientId?: string) => {
  const [response, setResponse] = useState<ClinicalResponse | null>(null);
  const [isStreaming, setIsStreaming] = useState(false);

  const startStream = async () => {
    const eventSource = new EventSource(
      `/api/v1/query/stream?query=${encodeURIComponent(query)}&patient_id=${patientId}`
    );
    
    eventSource.onmessage = (event) => {
      const data = JSON.parse(event.data);
      setResponse((prev) => ({ ...prev, ...data }));
    };
    
    eventSource.onerror = () => {
      eventSource.close();
      setIsStreaming(false);
    };
  };

  return { response, isStreaming, startStream };
};
